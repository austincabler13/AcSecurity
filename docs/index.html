AcSecurity

A Python module designed to scan applications for common security vulnerabilities, hardcoded secrets, and code quality issues.

Get Started

Features

Comprehensive Security Scanning

Common Vulnerability Checks

Scans for hardcoded secrets such as passwords or API keys in your code.

Dependency Vulnerability Checks

Uses pip-audit to identify known vulnerabilities in your installed Python packages.

Code Quality Checks

Uses pylint to identify code quality issues and ensure your code adheres to best practices.

Installation

Ensure you have Python 3.12.0 and pip installed on your machine.

pip install AcSecurity

Usage

After installing the module, you can use it to scan your application directory for vulnerabilities:

acsecurity /path/to/your/application

Example:

acsecurity /home/user/my_project
